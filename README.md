# Avaliador Escolar

Site: gerenciador de avaliações escolares (professores e alunos).

## Estrutura
- `index.html` — página inicial
- `avaliacoes.html` — gerenciamento de avaliações
- `alunos.html` — área do aluno
- `styles.css` — estilos
- `scripts/` — JS: `avaliacoes.js`, `alunos.js`
